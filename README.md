# Aplikacja do kursów walut

## Opis projektu

Celem projektu jest stworzenie aplikacji internetowej umożliwiającej pobieranie i wyświetlanie kursów walut z podziałem na lata, kwartały, miesiące i dni.
Projekt kładzie nacisk na testy automatyczne w podejściu BDD (Behavior-Driven Development) oraz na konteneryzację aplikacji z użyciem Dockera.

## Funkcjonalności

* Pobieranie kursów walut z API NBP
* Wyświetlanie danych z podziałem czasowym
* Testy automatyczne BDD
* Uruchamianie aplikacji w kontenerach Docker

## Technologie

* **Frontend:** Angular
* **Backend:** Django Rest Framework
* **Baza danych:** PostgreSQL
* **Konteneryzacja:** Docker
* **API:** NBP API ([https://api.nbp.pl](https://api.nbp.pl))
